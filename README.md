# EDHSwipe

### A Tinder-style commander discovery app for Magic: The Gathering EDH players

EDHSwipe helps you find your next commander by swiping through cards pulled live from the Scryfall API. Swipe right to save a commander, swipe left to skip, and filter by color identity to narrow down your options.

## Features

- Swipe yes or no on commanders
- Filter by color identity (W, U, B, R, G) or Converted Mana Cost
- View detailed commander info powered by Scryfall
- GSAP Animations

## Tech Stack

- **React**
- **Scryfall API**
- **GSAP**
- **Local Storage**
